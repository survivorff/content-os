# 想法捕获模板

复制到 `inbox/YYYY-MM-DD-slug.md`。写多少看你当时有多少想法，**最少一句话就行**。

---

```markdown
---
# 可选字段，全留空也行
platforms:
  # 你希望发的平台。留空让 scout 决定
  # - blog
  # - x
  # - xiaohongshu
  # - douyin

schedule:
  # ISO 8601 带时区。不写就用默认
  # blog: 2026-05-10T10:00:00+08:00
  # x: 2026-05-10T20:00:00+08:00

time_sensitive: false  # 刚爆出的新闻 / 热点 → true
---

# 我的想法

（自由写。不用整理。这是给 topic-scout 看的粗料。）

## 我为什么想写这个

<一句话动机。没有也行。>

## 我能提供的独家视角

<你的身份 / 经历 / 独家数据。越具体越好。>

## 核心观点

<如果已经想清楚，一句话概括。还没想清楚，写"待 scout 帮我定"。>

## 素材

<任何想到的素材：数据、链接、代码片段、亲身经历。列点就行。>
```

