# 抖音口播脚本风格指南

目标平台：**抖音**
受众：泛大众、短视频用户
形式：2-3 分钟口播视频

## 核心差异

抖音是"听觉 + 视觉"媒介，不是阅读媒介。脚本要是"说出来"而不是"看出来"。

## 3 分钟黄金结构

```
0-3 秒   │ 钩子（Hook）     │ 让人停下来
3-15 秒  │ 价值承诺         │ 你能得到什么
15-120 秒│ 主体内容         │ 3-5 个要点
120-180  │ 总结 + CTA       │ 关注 + 下期预告
```

**最重要的是前 3 秒**。如果前 3 秒抓不住人，后面再精彩都没用。

## 钩子模板（前 3 秒）

### 反常识型

> "Solana 上有个赌场，赌客都不知道自己在给庄家打工。"

### 数据震撼型

> "一个网站 2 年赚了 10 亿美金，99% 的人没听过。"

### 悬念型

> "这张图，90% 的交易员都看错了，你呢？"

### 身份型

> "我在交易所做了 4 年后端，这 3 个观察颠覆了我。"

### 冲突型

> "很多人说 Web3 是骗局，但我一年赚到了 10 年的工资。"

## 主体结构

把长文拆成 **3-5 个"说得完的"要点**：

```
要点 1（30 秒内）
- 一个观察
- 一个支撑（数据 / 例子）
- 一句结论

要点 2（30 秒内）
...
```

每个要点一个"小高潮"。不要拖。

## 语言特征

### 口语化，不是书面语

**书面（不好）：**
> "这个市场的参与者可以分为四个层级，形成了一个完整的食物链。"

**口语（好）：**
> "这个圈子里，能挣钱的就 4 种人，其他都是韭菜。"

### 短句

每句话 10-15 字最佳。超过 20 字，听众会走神。

**坏：** "在 Pump.fun 这个平台上，任何人只需要花不到两美元的成本，就能在 30 秒内创建出一个新的加密货币。"

**好：** "在 Pump.fun 上，发一个币，花 2 美元，30 秒搞定。"

### 有节奏感

多用：
- 三字短语（"快、准、狠"）
- 对仗句（"有人赚，有人亏"）
- 反问（"真的吗？"）
- 停顿（"..."）

### 避免

- ❌ 书面词（然而、因此、综上）
- ❌ 长从句
- ❌ 专业术语（必须用要解释）
- ❌ 被动句

## 视觉提示（脚本里标注）

因为是口播视频，脚本要标注"说这句时屏幕显示什么"：

```
[说的话]
（视觉：展示什么）
```

示例：

```
"Pump.fun 两年赚了 10 亿美金。"
（视觉：10 亿美金的数字大写，快速放大）

"这是什么概念？"
（视觉：和其他 APP 的对比图）

"相当于两年前的抖音。"
（视觉：抖音的截图 + 对比）
```

## 长度

- 2 分钟视频：脚本约 400-600 字
- 3 分钟视频：脚本约 600-900 字

**不要超过 3 分钟**。抖音完播率是核心指标。

## 必备元素

### 开场必有

- 3 秒内的钩子
- 一句"我是谁"的身份

### 结尾必有

```
"关注我 @xxx，下期讲 [预告]。"
（视觉：关注按钮 + 下期 Teaser）
```

### 全程注意

- 表情丰富（但不要过）
- 手势自然
- 节奏不拖

## 不要做的事

- ❌ 不要搬运博客原文读
- ❌ 不要照 ppt 念
- ❌ 不要太长（超 3 分钟完播率崩）
- ❌ 不要复杂名词不解释
- ❌ 不要没 hook 就开始讲

## 示例（完整短脚本）

```
【标题】：在交易所做 Web3 基建，赚得比我爸 10 年还多

[0:00-0:03] 钩子
"在交易所做了 4 年 Web3，赚得比我爸 10 年还多。"
（视觉：工资单前后对比图，打码）

[0:03-0:10] 价值承诺
"但别急着转行。今天告诉你 3 个真相，
说完你自己决定要不要来。"
（视觉：3 个数字 1、2、3 闪现）

[0:10-0:40] 真相 1
"真相 1：钱多，但赚起来心惊胆战。
一个 bug 可能让公司损失几十万美金。
不是互联网那种可以容错的环境。"
（视觉：Bug 事件新闻截图）

[0:40-1:10] 真相 2
"真相 2：学不完。
传统后端你学完框架就能干 5 年。
Web3 每 3 个月一个新叙事，
你不持续学就被淘汰。"
（视觉：技术名词轮播）

[1:10-1:40] 真相 3
"真相 3：行业周期非常快。
牛市你觉得自己是天才，
熊市看自己就像傻子。
心理素质差的劝退。"
（视觉：K 线图起伏）

[1:40-1:55] 总结
"所以如果你符合：
能抗压 + 爱学 + 有技术基础，
来。
不然，再想想。"

[1:55-2:00] CTA
"关注我，下期讲具体怎么上手。"
（视觉：关注引导）
```

## 发布时间

抖音的活跃时段：
- 中午 12:00-14:00
- 晚上 19:00-23:00
- 周末全天都可以

数据分析类内容，工作日晚上 20-22 点最好。
