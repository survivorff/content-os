# CTA 模板（最后一条）

最后一条的唯一任务是**导流到博客 + 提高互动**。不要再塞论点。

---

## 标准模板

```
完整文章（含 [关键词]）：
blog.frankfu.cloud/posts/[slug]

[一句互动]

@FrankFu2262
```

**示例：**

> 完整文章（含 RPC 优化代码）：
> blog.frankfu.cloud/posts/solana-rpc-optimization
>
> 你们在生产环境踩过最恶心的 RPC 坑是哪个？
>
> @FrankFu2262

---

## 互动一句话的模板

按 Thread 类型选：

### 技术类 Thread

- "你们在生产环境踩过最恶心的 XX 是哪个？"
- "有更好的方案吗？评论区讨论。"
- "下一篇想看哪个方向：A / B / C？"

### 观点类 Thread

- "你认同吗？评论区见。"
- "反对意见我都想看看。"
- "不同看法的欢迎 quote 反驳。"

### 故事类 Thread

- "你有过类似经历吗？"
- "评论区讲讲你的版本。"

### 清单类 Thread

- "你会加一条什么？"
- "有没有第 N+1 条我漏了？"

**原则**：互动问题要**具体、开放、容易答**。不要问"你觉得呢？"这种没指向的。

## 禁止

- ❌ 最后一条只有链接，没互动引导
- ❌ 塞 5+ 个 hashtag
- ❌ 求转发（"求扩散"、"麻烦转发"）—— low
- ❌ 放多个链接（博客 + 小红书 + 抖音）—— 分散注意力
- ❌ "关注我不迷路" 这种套话

---

## 字符数

CTA 条一般会比较短，50-150 字。不要硬撑到 260。短结束有力量。
