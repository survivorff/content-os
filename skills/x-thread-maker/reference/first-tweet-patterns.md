# 第一条推文的模板

这是 Thread 最关键的一条。花 30% 的精力在这里。

---

## 评判标准

一个好的第一条应该让读者 3 秒内产生 2 个反应之一：

1. **"诶，真的吗？"** —— 反常识
2. **"我要看看他怎么说"** —— 身份或故事钩

做不到上面任何一个，就失败。

---

## 模板 1 — 反常识断言

```
[大家都以为的事实 A]

[其实是 B，附一个数字]

作为 [身份]，下面讲 [N] 个被忽略的 [事]。

🧵
```

**示例：**

> Base、BSC、以太坊都在做 Meme，但 Solana 占了 90%+ 市场。
>
> 不是"便宜+快"那么简单。
>
> 作为在交易所做 2 年 meme 基建的人，告诉你是哪三个被忽略的因素。
>
> 🧵

---

## 模板 2 — 数据震撼

```
[一个震撼的数字]

[这个数字的意义，用对比]

[下文要讲的是]

🧵
```

**示例：**

> $10.3B。
>
> 这是 Pump.fun 2024 年的累计收入。用了 18 个月。团队 10 个人。
>
> 作为做过 2 年平台方的人，下文复盘这是怎么做到的。
>
> 🧵

---

## 模板 3 — 故事开场

```
[具体时间 + 事件]

[一个转折或结果]

[下文要讲的是 N 个从中抽出的观察]

🧵
```

**示例：**

> 2024 年 3 月凌晨 3 点，我们交易所的撮合引擎卡了。
>
> 罪魁祸首是一个看起来人畜无害的 Solana RPC 调用。
>
> 下面讲生产级 Solana dApp 的 5 个 RPC 地雷，每一个我都踩过。
>
> 🧵

---

## 模板 4 — 身份 + 清单预告

```
[我是谁，做了 X 年 Y]

[发现的 N 条反常识 / 经验]

[一条比一条狠 / 我全踩过]

🧵
```

**示例：**

> 在交易所做 Web3 后端 2 年。
>
> 踩出 5 条反常识经验，和 Web2 完全相反。
>
> 第 3 条让我交了 $50k 学费。
>
> 🧵

---

## 模板 5 — 问题开场（慎用）

```
[一个具体的读者场景 / 痛点]

[本文给答案]

🧵
```

**示例：**

> 上线 Solana dApp 第 3 天，RPC 成本飙到 $5k/月。
>
> 如果你也遇到这种情况，下面有 4 个立竿见影的优化点。
>
> 🧵

**注意**：问题型开场弱于其他模板，只用在教程类 Thread。

---

## 绝对不要这样开第一条

- ❌ "今天给大家分享一下 XXX"
- ❌ "很久没写 Thread，今天来聊聊"
- ❌ "最近看到一个有意思的..."
- ❌ 纯链接 + "must read"
- ❌ 一段文学散文
- ❌ 超过 260 字（被截断看不到 🧵）

---

## 字符数

- 第一条严格 **< 180 字**（中文 2 字符规则）
- 给 🧵 和换行留空间
- 宁可砍到 150，不要贴到 260

---

## 从 outline 选哪个模板

| outline 推荐的 hook | 对应 Thread 模板 |
|---|---|
| 数据钩 | 模板 2 |
| 故事钩 | 模板 3 |
| 反差钩 | 模板 1 |
| 清单型文章 | 模板 4 |
| 技术教程型 | 模板 5（慎用） |

如果拿不准，**默认用反常识断言（模板 1）**——最通用、数据最稳。
