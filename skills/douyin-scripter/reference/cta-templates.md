# 抖音 CTA 模板

视频最后 5-10 秒。

---

## 标准模板

```
"[一句话总结]

关注我 @[账号名]，下期讲 [预告]。"

（视觉：关注按钮 + 下期 Teaser）
```

**示例：**

> "所以：能抗压 + 爱学 + 有技术基础，来。不然，再想想。
>
> 关注我，下期讲 Web3 后端具体怎么上手。"
>
> （视觉：关注按钮放大 + 下期标题卡）

---

## CTA 变体

### 系列预告型

```
"这是《[系列名]》第 [N] 集。
下集讲 [预告]。
关注我不错过。"
```

### 引导互动型

```
"你转 Web3 的顾虑是什么？评论区告诉我。
我看到都会回。"
```

### 关注 + 私信

```
"想具体聊的，主页加我。
关注我，更多真实经验陆续发。"
```

---

## CTA 必须明确

- 要不要关注？
- 要不要评论？
- 要不要看下一期？

**只做 1-2 个 call to action**。贪多就一个都做不到。

通常：**关注 + 下期预告** 是最佳组合。

---

## 禁止

- ❌ "求三连"（low）
- ❌ "点赞超过 1000 我就出下一期"（操纵用户，抖音限流）
- ❌ 塞多个外链账号（"我的微信是...小红书是..."）—— 抖音会封
- ❌ 卖货 / 导流到付费
- ❌ 没有 CTA，戛然而止

---

## 时长

CTA 总时长 5-15 秒。不要拖。

**节奏：**

```
[1-2 句总结（5-8 秒）]

[CTA（5-10 秒）]
```

---

## 引导关注的小技巧

- **指向性手势**：说"关注我"时用手指向屏幕左下（抖音关注按钮位置）
- **视觉强化**：关注按钮放大 / 高亮
- **具体预告**：下期的标题要具体吸引人，模糊的 teaser 没用
