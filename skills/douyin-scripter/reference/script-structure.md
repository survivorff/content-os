# 3 分钟口播结构

抖音视频的黄金结构。

---

## 时间分配

| 时间段 | 内容 | 字数 |
|---|---|---|
| 0-3 秒 | Hook | 20-30 字 |
| 3-15 秒 | 价值承诺 | 60-80 字 |
| 15-120 秒 | 主体（3-5 个要点） | 400-600 字 |
| 120-180 秒 | 总结 + CTA | 80-120 字 |

总字数：600-900 字。

---

## 各段的目标

### 0-3 秒 Hook

**目标**：让滑手指的人停下来。

方法：
- 反常识断言
- 震撼数字
- 身份自曝
- 冲突画面

见 `hook-templates.md`。

### 3-15 秒 价值承诺

**目标**：告诉观众"我接下来要给你什么"。

模板：

```
[我是谁 + 有什么资格说]

[这个视频你能得到什么]

[用 N 个要点的形式]
```

**示例：**

> 在交易所做 Web3 后端 2 年。
>
> 今天告诉你 3 个真相，决定要不要转行。

### 15-120 秒 主体

**目标**：交付承诺，让观众觉得"没白看"。

3-5 个要点：
- 每个要点 25-40 秒
- 先给结论，后给证据（抖音倒序讲更抓人）
- 每个要点之间用"所以..."、"但是..."衔接

**结构：**

```
要点 1（25-40 秒）
- 结论句
- 证据 / 例子
- 小收束

要点 2（25-40 秒）
- 结论句
- 证据 / 例子
- 小收束

要点 3（25-40 秒）
- 结论句
- 证据 / 例子
- 小收束
```

### 120-180 秒 总结 + CTA

**目标**：巩固印象 + 引导关注。

```
[一句话总结 / 金句]

[CTA：关注 + 下期预告]
```

---

## 不同长度的视频

### 1 分钟短视频（300 字）

更极简的结构：

```
0-3 秒   Hook
3-10 秒  价值承诺
10-50 秒 主体（只放 1-2 个要点）
50-60 秒 CTA
```

用于"小话题"或"测试爆点"。

### 2 分钟（600 字）

标准结构，主体 3 个要点。

### 3 分钟（900 字）

标准结构，主体 4-5 个要点。

**不要超过 3 分钟**。数据证明完播率暴跌。

---

## 节奏控制

- **前 15 秒密度要高**（信息 + 情绪）
- **中间要有 1-2 个"扣"**（停顿、反问、语速变化）
- **倒数 15 秒再起一个小高潮**（防止观众最后分心）

脚本里可以标记：

```
[停顿 1 秒]
[语速快]
[语速慢]
[表情：皱眉]
```

给作者录制时参考。
