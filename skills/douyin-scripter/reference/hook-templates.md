# 抖音 Hook 模板（前 3 秒）

前 3 秒是整个视频最重要的部分。Hook 失败，后面再精彩也没用。

---

## 5 个高转化 Hook 模板

### 1. 反常识断言

```
"[大家都以为的事] 其实是 [反面]。"
```

**示例：**
- "Solana 上有个赌场，赌客都不知道自己在给庄家打工。"
- "Web3 后端赚的多？错，心理成本你没算。"

### 2. 震撼数字

```
"[一个数字]。[这是什么]。"
```

**示例：**
- "10 亿美金。一个网页应用 2 年赚到的。99% 的人没听过。"
- "50 万美金。我们差点因为一行代码赔进去。"

### 3. 身份自曝

```
"我 [做了什么 + 时长]。"
```

**示例：**
- "我在交易所做 Web3 后端 4 年，这 3 个观察颠覆了我。"
- "我转 Web3 1 年，多赚了 10 年工资。但不推荐你来。"

### 4. 冲突 / 悬念

```
"[具体场景] + [冲突点]。"
```

**示例：**
- "凌晨 3 点，我们交易所的撮合引擎卡了。罪魁祸首只有一行代码。"
- "这张图，90% 的交易员都看错了。你呢？"

### 5. 直击痛点

```
"如果你 [读者场景]，[本视频给答案]。"
```

**示例：**
- "如果你在考虑转 Web3，这 3 个真相你得先知道。"
- "如果你的 Solana RPC 月费超过 $1000，这 4 条能帮你省一半。"

---

## Hook 的铁律

1. **3 秒内讲完**。20-30 字。播音速度每秒 10 字左右。
2. **不要疑问开场**（弱）。
3. **不要"大家好"**（超弱）。
4. **第一个字要有冲击**。避免"今天"、"最近"、"其实"开头。
5. **具体 > 抽象**。"10 亿美金" > "巨额收入"。"凌晨 3 点" > "某天"。

---

## 视觉配合 Hook

Hook 必须配合强视觉。建议：

| Hook 类型 | 视觉 |
|---|---|
| 反常识 | 两张对比图（你以为的 vs 实际） |
| 震撼数字 | 数字大字 + 快速放大 |
| 身份自曝 | 工牌 / 证件 / 打码工资单 |
| 冲突 | 新闻截图 / 代码截图 |
| 痛点 | 错误提示 / 账单截图 |

视觉提示必须在脚本里明确写出来（见 `script-output-format.md`）。

---

## 从博客 hook 改到抖音 Hook

博客 hook 往往 50-100 字。抖音 Hook 必须压到 20-30 字，只保留最震撼的部分。

**示例：**

博客 hook：
> 2024 年 1 月，Pump.fun 上线。当时没什么人认真对待。18 个月后，它成了 Solana 上第一个累计收入破 $10B 的应用。作为在里面做了 2 年平台方的人，我想把这个过程复盘一下。

抖音 Hook（压到 3 秒）：
> "一个网页应用 2 年赚了 $10B。99% 的人没听过。"

抖音版本砍掉了：时间、作者身份、目的。只保留**最强的数据反差**。身份和目的放在 3-15 秒的"价值承诺"段落。

---

## 不要做

- ❌ Hook 超过 30 字
- ❌ Hook 里有专业术语没解释
- ❌ Hook 卖关子到第 10 秒还没揭晓
- ❌ "大家好我是 XXX" 类的开场
- ❌ Hook 和内容不符（诈骗式标题党，完播率会惩罚）
