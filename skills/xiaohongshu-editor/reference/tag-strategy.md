# 标签策略

小红书靠标签分发。一个好标签组合能让内容翻倍触达。

---

## 标签组合公式

```
5-10 个标签 = 2-3 个垂直 + 2-3 个身份 + 2-3 个泛流量
```

---

## 垂直标签（话题本身）

- #Web3
- #区块链
- #Solana
- #以太坊
- #加密货币
- #DeFi
- #MEME
- #NFT

**规则**：至少 2 个，不要超过 4 个。垂直标签太多会限制分发到小圈子。

---

## 身份标签（作者的身份）

- #程序员
- #工程师
- #后端开发
- #互联网
- #大厂
- #互联网人
- #职场

**规则**：用 2-3 个。**让读者能一秒对号入座**。

---

## 泛流量标签（吸引跨圈）

- #副业
- #认知升级
- #自我提升
- #干货
- #学习
- #科技
- #创业
- #AI

**规则**：用 1-3 个。**选和内容真的相关**的，别硬凑。

---

## 按内容类型的标签模板

### 技术深度内容

```
#Web3 #区块链 #Solana #程序员 #后端开发 #技术分享 #干货 #互联网
```

### 转型故事

```
#Web3 #程序员 #职场 #转型 #互联网 #副业 #认知升级 #大厂
```

### 行业观察

```
#Web3 #加密货币 #科技 #行业观察 #认知升级 #趋势 #赚钱
```

### 工具 / 指南

```
#Web3 #工具 #程序员 #效率 #学习 #干货 #技术分享
```

---

## 禁止

- ❌ 超过 10 个标签（分发算法反而降权）
- ❌ 全是垂直标签（没泛流量）
- ❌ 全是泛标签（没锚点）
- ❌ 硬凑热门标签（"#情感"、"#恋爱"这种不相关的）
- ❌ 打错字 / 错别字标签

---

## 冷启动 vs 长期

**冷启动时（账号粉丝 < 1000）**：

多用泛标签，蹭大流量。比例：2 垂直 + 2 身份 + 4 泛。

**稳定期（粉丝 > 5000）**：

多用垂直标签，巩固垂直受众。比例：4 垂直 + 2 身份 + 2 泛。
